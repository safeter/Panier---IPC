# 🛒 Panier — Prix d'épicerie au Québec

PWA mobile de suivi de prix d'épicerie au Québec.

## 🔗 Application
**https://safeter.github.io/Panier---IPC/**

## Fonctionnalités
- 📷 Scanner de reçu via Claude Vision
- 📊 IPC personnel — votre inflation réelle depuis le 1er achat
- 🔴 Alertes prix élevés (+15% vs médiane)
- 🟢 Alertes prix plancher historique  
- ⚠️ Détection shrinkflation automatique
- 💲 Prix normalisé au 100g/100ml
- 🇶🇨 Données filtrées par région (code postal)
- ☁️ Backend Supabase — données privées par défaut

## Stack
HTML/CSS/JS vanilla · PWA · Claude API · Supabase

## Installation mobile
**iPhone** : Safari → Partager → Sur l'écran d'accueil  
**Android** : Chrome → ⋮ → Installer l'application

## Données
814 transactions · Déc 2023 – Jan 2026  
Super C, Maxi, Provigo, IGA, Adonis
